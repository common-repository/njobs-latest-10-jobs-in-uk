== Description ==

Njobs widget displays 10 latest vacancies from the United Kingdom on your website. Njobs users are able to add their own vacancies on the website. This widget presents their ten latest vacancies. Vacancies come from Njobs users, so you have 100 per cent certainty the content is unique. It makes Njobs widget valuable tool, and that is why everyone should be interested in putting it on their websites.

== Installation ==

1. Upload widget_njobs_lastjob.php to /wp-content/plugins/.
2. Activate plugin through the WordPress Plugins menu.
3. Activate widget at the Appearance > Widgets menu in WordPress.
4. Set display options from the widget's administration panel.
